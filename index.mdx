## What is this project?

The **Prime Number Neural Network** is a research project that explores whether an AI can learn to identify prime numbers without performing trial division. Numbers are encoded as **multi-dimensional entities** — each integer is transformed into a rich vector of arithmetic features before being classified by a two-stage machine learning ensemble.

The key insight: primes leave a measurable *geometric and structural footprint* across base systems, remainder patterns, and binary representations. By surfacing these hidden dimensions, an ML model can achieve **100% precision** (zero false positives) while extrapolating far beyond its training range.

<Note>
  The model was trained only on integers 1–10,000 and tested on completely unseen ranges. **100% precision** means it never mislabels a composite as prime across any range tested.
